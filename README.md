# **Real-Time Collaborative Whiteboard**

A web-based whiteboard application that allows multiple users to draw, collaborate, and see each otherâ€™s changes in real time. The application features real-time synchronization, undo/redo functionality, and the ability to export drawings as an image.

---

## **Features**

- **Real-Time Collaboration:** Users can draw and see updates in real time across multiple tabs or devices.
- **Undo/Redo Functionality:** Revert or restore previous drawing actions.
- **Clear Canvas:** Clear the entire whiteboard for all users in real time.
- **Export as Image:** Download the current drawing as a PNG file.
- **User Count Display:** Shows the number of connected users.

---

## **Technologies Used**

- **Frontend:**
  - React.js
  - HTML5 Canvas
  - Socket.IO (client)
- **Backend:**
  - Node.js
  - Express.js
  - Socket.IO (server)
